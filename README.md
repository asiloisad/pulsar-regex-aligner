# regex-aligner

Align text by regex statment.

## Installation

The official [Atom packages store](https://atom.io/packages) has been disabled. To get latest version run the shell command

    apm install bacadra/atom-regex-aligner

and obtain the package directly from Github repository.

The package has compability with [Pulsar](https://pulsar-edit.dev/) and can be install

    pulsar -p install bacadra/atom-regex-aligner
